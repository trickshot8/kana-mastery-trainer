# 🎯 Kana Mastery Trainer

A focused kana training tool designed to bridge the gap between **“familiarity” and “accurate recall.”**

Built from a real learning need — for learners who have seen hiragana/katakana before but struggle to remember them precisely.

---

## ✨ Why this exists

Many tools (like Duolingo) help you get familiar with kana, but often leave you stuck in this stage:

> "Looks familiar, but can't recall instantly."

This tool is designed specifically to solve that problem.

---

## 🚀 Features

- 🎯 **Customizable practice sets**  
  Practice only the kana you want (e.g. a specific row)

- 🔁 **Bidirectional training**  
  - Kana → Romaji  
  - Romaji → Kana  
  - Mixed mode  

- ❗ **Mistake-driven reinforcement**  
  Focus on what you get wrong

- ⚡ **High-frequency drills**  
  Designed for short, intensive sessions (not long-term spaced repetition)

- 🧠 **Mastery-focused design**  
  No gamification, no streaks — just efficient learning

---

## 🧭 Who is this for?

This tool is best for:

- Learners who already have **basic exposure to kana**
- People stuck in the **“almost memorized” stage**
- Anyone who wants to **master kana in 1–2 days of focused practice**

## 🧠 Design Philosophy

This is not a full language learning app.

It is a **“learning bottleneck breaker”** —  
a tool designed to help you move from: recognition → accurate recall → automatic response
